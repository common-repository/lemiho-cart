== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/lemihocart` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Product settings->Settings of plugin comfim something sytem of Lemiho Cart
4. Us not require pay but if you can then pay into me 3$ for version 1.0, i will support 24 / 7 ( Languages: english / Viet Nam )
